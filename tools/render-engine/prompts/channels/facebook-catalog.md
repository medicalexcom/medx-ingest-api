# Channel: Facebook Catalog / Shop

This channel is short-form, mobile-first, and plain-text friendly.

## Output Constraints
- Prefer short, punchy copy.
- Catalog description target: ~200 chars.
- Shop description may extend up to ~500 chars when needed.
- No HTML requirement.
- No internal links/manuals sections.
- Focus on top features and buyer relevance.

## Tone
- Crisp, social-friendly, and factual.
- May allow light emoji if brand-safe and profile permits.

## Content Rules
- Do not invent specs.
- Keep claims evidence-backed.
- Avoid medical/regulatory statements unless directly sourced.
